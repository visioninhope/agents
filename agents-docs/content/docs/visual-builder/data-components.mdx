## How to create a data component

1. Go to the Data Components tab in the left sidebar. Then select "New data component".
2. Add in an id, name, description, and enter in a JSON schema. These are required fields.
3. Click "Submit".

To visually add the data component to the graph, see the [Graphs](/visual-builder/graphs#adding-data-components) page for details.
